# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-01-21

### Added
- **Premium Glassmorphism UI**: Completely redesigned dashboard with high-contrast dark theme and fluid animations.
- **Universal Paste Support**: Ability to paste images directly from the clipboard (**Ctrl+V**) across Web, Desktop, and Extension.
- **Master Shortcuts Panel**: Integrated keyboard shortcut overlay for power-user workflows.
- **Dynamic Image Counter**: Real-time badge in the header tracking the current processing queue.
- **Factory Presets**: New expansion packs for iOS Icons, Android Icons, and Social Media profiles.
- **Security Policy**: Official `SECURITY.md` outlining our local-processing commitment.
- **Publishing Guide**: Detailed guide for Chrome Web Store deployment.
- **High-Res Promo Assets**: New small and large promotional tiles for store listings.

### Changed
- Refactored core conversion logic for better performance during large batch exports.
- Synchronized feature sets across the Web, Desktop (Electron), and Chrome Extension platforms.
- Updated documentation (README) to reflect "Pro" branding and architecture changes.

### Fixed
- Resolved theme toggle persistence bugs where preferences were lost after page reload.
- Fixed duplicated event listeners in the Desktop app initialization.

## [1.0.0] - 2025-10-15
- Initial release with basic batch conversion and resizing capabilities.
