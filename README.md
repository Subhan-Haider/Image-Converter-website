**Q: Is my data safe?**

## ğŸ› Troubleshooting

### Extension Not Working
1. Refresh the page after installing the extension
2. Check if the extension is enabled in `chrome://extensions/`
3. Try reloading the extension

### Page Scanner Not Finding Images
1. Ensure you're not on a restricted page (chrome://, edge://, etc.)
2. Some websites use lazy loading - scroll down first
3. Check browser console for errors

### Conversion Fails
1. Verify the image file isn't corrupted
2. Try a smaller batch size
3. Check available browser memory
4. Disable browser extensions that might interfere
